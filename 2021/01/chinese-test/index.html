win10本地博客搭建整理

2 一月 2021 发布在 tranquilpeak, features

采用：hugo + github + aliyun

​ ！！！前提条件：已安装 GIT，并熟悉git的相关操作

hugo:

一：官网下载hugo，地址：https://github.com/gohugoio/hugo/releases

​ 我们根据自己的系统来选择版本即可，笔者是win10x64位。

​ 因为github的下载太慢，可以通过gitee下载。

第一步：

第二步：

第三步(复制到gitee后直接下载即可)：

二：在本地磁盘建文件夹

​ ①：在D盘新建一个文件夹：D:\Hugo ps：不一定要在D盘新建，D、E、F盘都行

​ ②：在Hugo目录下面再分别创建两个子文件夹：D:\Hugo\bin 、D:\Hugo\Sites

三：解压hugo安装包

​ ①：解压之后得到三个文件hugo.exe、LICENSE、README.md；

​ ②：把hugo.exe放进 D:\Hugo\bin文件夹；

四：配置环境变量

​ 对着 ”我的电脑“ 右键属性，在高级里面，选择环境变量，在系统变量里面找到path，点击“编辑”，

​ 在新弹出的窗口里，点击“新增”，把 D:\Hugo\bin 复制，然后保存退出。

​ ps：笔者这里的路径是 D:\Hugo\bin ，请根据自己的bin目录进行修改

五：部署hugo

​ ①： 打开 D:\Hugo\Sites目录， 在目录栏 输入 cmd ，回车；

​ ②： 在弹出的cmd窗口输入指令：D:\Hugo\Sites> hugo new site zhonger.com

​ ps: “zhonger.com” 是新的网站名称，可根据自己的喜好修改

​ 命令输入完毕后，如果一切顺利，在sites目录里面生成zhonger.com的文件夹；

​ ③： 继续输入指令：cd zhonger.com 进入zhonger.com文件夹；

​ 继续输入指令： hugo new about.md 该指令会到 content 目录下自动生成 about.md文件；

​ ④： 在zhonger.com 的目录下(若不是该目录，则回到此目录) 输入指令：hugo new post/first.md

​ ps：该步骤可以省略

​

六：安装皮肤

​ ①：输入指令：cd themes 进入themes目录

​ ②： 输入指令： git clone git://github.com/dim0627/hugo_theme_aglaus aglaus 克隆一份皮肤

​ ps：皮肤的官网是：https://themes.gohugo.io/ 可以从上面选择自己喜欢的皮肤，

​ clone时记得把地址的http改成 git，否则会clone失败

​ ③：到 D:\Hugo\Sites\zhonger.com\themes\自己下载的主题名称\exampleSite 目录，把该目录下的

​ config.toml文件 复制到 zhonger.com目录下

七：运行hugo

​ ①：输入指令：hugo server 若出现下图的文字，则说明，启动成功

​ ②： 用 http://localhost:1313/ 来访问自己搭建的博客。

​

github:

第一步：在自己的github账号上新建一个仓库，名字是：自己账号名称的小写.github.io

​ 笔者账号ID：zhonger94，仓库名称：zhonger94.github.io

第二步： 按 ctrl+c 停止hugo，输入指令:

​ hugo –theme=hugo-tranquilpeak-theme –baseUrl=“https://zhonger94.github.io/" –buildDrafts

​ 会在当前zhonger.com目录生成一个public文件夹

​ ps：baseUrl所填写的地址格式是：http://自己仓库的名字/

第三步： 在自己本地磁盘任意目录，打开git软件，输入指令: git clone 自己仓库的SSH地址（或者https地址）

​ 打开 clone 得到的 文件夹，复制里面的 .git文件夹到 public文件夹里

第四步： 在public文件夹 里打开 git软件，输入指令

​ ①：git add . #提交到暂存区

​ ②：git commit -m “备注的信息” #提交到本地库

​ ③： git push #提交到github

​ 通过以上指令把本地的hugo上传到github仓库

第五步: 打开浏览器，输入 https://zhonger94.github.io/ ，即可访问自己的博客,无需在cmd窗口输入指令 hugo server

# Aliyun:

第一步：登录aliyun官方网站，查找自己喜欢的域名，付款购买

​

第二步：解析域名

第三步：github修改仓库的 setting，往下拉，找到Custom domain，输入自己购买的域名，点击保存

第四步：在浏览器输入自己购买的域名，即可访问博客。

最后：把自己想要发表的文章放进 D:\Hugo\Sites\zhonger.com\content\post 目录下即可

​ ps：文件需要.md格式，模板可参考

​ D:\Hugo\Sites\zhonger.com\themes\hugo-tranquilpeak-theme\exampleSite\content\post目录下 的文件

注意：每次更新博客后，都需要重新打包public 并上传到github 。

附上配置文件 config.toml 的相关注解：

baseURL = “https://your_name.github.io/your_name/"
