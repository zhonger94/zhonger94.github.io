babel转码器的使用

19 一月 2019 发布在 tranquilpeak, features

​ 详情点我

前提：已安装转码工具

在命令行进行：

①查看版本：babel –version

②若没有安装，则用下面的命令安装：

​ npm install –global babel-cli

使用

1、初始化项目

在命令行输入：npm init -y

2、创建文件

//文件内容可变，这里是举例

// 转码前
// 定义数据
let input = [1, 2, 3]
// 将数组的每个元素 +1
input = input.map(item => item + 1)
console.log(input)

3、配置.babelrc (文件的全名就是 ：.babelrc )

//presets字段设定转码规则，将es2015规则加入 .babelrc：
//该文件要创建在工作区目录下
{
    "presets": ["es2015"],
    "plugins": []
}

4、安装转码器 （只在当前项目有效）

在命令行输入: npm install –save-dev babel-preset-es2015

5、转码

下面的操作在命令行进行：

####  方法一：###
# 转码结果写入一个文件
mkdir dist1
# --out-file 或 -o 参数指定输出文件
babel src/example.js --out-file dist1/compiled.js


####   方法二：
# 只转换单个文件   ###  常用 ##############
babel src/example.js -o dist1/compiled.js

# 整个目录转码
mkdir dist2
# --out-dir 或 -d 参数指定输出目录
babel src --out-dir dist2

# 或者 整个目录转码  ####### 常用 ######################
babel src -d dist2
