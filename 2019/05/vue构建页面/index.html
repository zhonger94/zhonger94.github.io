vue构建页面

12 五月 2019 发布在 tranquilpeak, features

​ 详情点我

一、创建html页面，使用快捷键生成html代码

快捷键为：！

二、引入vue的js文件，类似jquery

① 把 vue的js文件复制到工程的磁盘目录

② 引入

<script src="vue.min.js"></script>

三、在html页面创建div标签，div添加idshux

四、编写vue代码 （固定结构）

五、使用插值表达式，获取data(变量名)的值

<body>
    <!-- 三 -->
    <div id="app">
        <!-- 五 {{}} 差值表达式，绑定vue中的data数据 -->
        {{message}} <!-- vue对象中 data属性的变量名 -->
    </div>

    <!-- 二 -->
    <script src="vue.min.js"></script>
    
    <!-- 四 -->
    <script>
        //创建一个vue对象，固定写法
        new Vue({
            el:'#app',//绑定vue作用域，值为 div标签的id，固定写法
            data:{
                message:'hello vue!' //该行可以自定义
            }
        })
    </script>
</body>

六、在vs code中创建代码片段：

文件 => 首选项 => 用户代码片段 => 新建全局代码片段/或文件夹代码片段：

vue-html.code-snippets 注意；vue可改变，其他固定

​ 注意：

​ 制作代码片段的时候，字符串中如果包含文件中复制过来的“Tab”键的空格，要换成“空格键”的空格。

{
        "vue htm": {
            "scope": "html",
            "prefix": "vuehtml",
            "body": [
                "<!DOCTYPE html>",
                "<html lang=\"en\">",
                "",
                "<head>",
                "    <meta charset=\"UTF-8\">",
                "    <meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">",
                "    <meta http-equiv=\"X-UA-Compatible\" content=\"ie=edge\">",
                "    <title>Document</title>",
                "</head>",
                "",
                "<body>",
                "    <div id=\"app\">",
                "",
                "    </div>",
                "    <script src=\"vue.min.js\"></script>",
                "    <script>",
                "        new Vue({",
                "            el: '#app',",
                "            data: {",
                "                $1",
                "            }",
                "        })",
                "    </script>",
                "</body>",
                "",
                "</html>",
            ],
            "description": "my vue template in html"
        }
    }
