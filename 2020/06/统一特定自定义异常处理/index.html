统一、特定、自定义异常处理

21 六月 2020 发布在 tranquilpeak, features

​ 详情点我

一、统一异常处理类

创建一个异常处理类，并添加规则

@ControllerAdvice
public class GlobalExceptionHandler {

    //指定出现什么异常执行该方法： Exception.class 表示所有异常
    @ExceptionHandler(Exception.class)
    @ResponseBody //为了返回数据
    public R error(Exception e) { // R：自定义类
        e.printStackTrace();
        return R.error().message("执行了全局异常处理...");
    }
}

二、特定/特殊异常处理类

在异常处理类中添加规则

@ControllerAdvice
public class GlobalExceptionHandler {
    //其他的异常处理类.......   
    
    //特定异常
    @ExceptionHandler(ArithmeticException.class)
    @ResponseBody //为了返回数据
    public R error(ArithmeticException e) { // R：自定义类
        e.printStackTrace();
        return R.error().message("执行了特定异常处理...");
    }
}

三、自定义异常处理类

1、创建一个继承RunTimeException的类

@Data
@AllArgsConstructor
@NoArgsConstructor
public class GuliException extends RuntimeException {
    private Integer code;//状态码
    private String msg; //异常信息
}

2、在统一异常类添加规则

@ControllerAdvice
public class GlobalExceptionHandler {
    //其他的异常处理类.......   
    
     //自定义异常
    @ExceptionHandler(GuliException.class)
    @ResponseBody //为了返回数据
    public R error(GuliException e) { // R：自定义类
        e.printStackTrace();
        return R.error().code(e.getCode()).message(e.getMsg());
    }
  
}

3、在代码中应用自定义异常

public void testException() {
      //模拟异常
        try {
            int i = 10 / 0;
        } catch (Exception e) {
            throw new GuliException(20001, "自定义异常处理...");
        }
}
